Resource Guide

A quick reference to basic HTML and CSS code!

Covering tags, explanations and stylings to make your website functional and have backbone.

HTML stands for Hypertext Markup Language and uses tag-based code to describe the structure of a website/webpage.

CSS stands for Cascading Style Sheets and worries about the format and visuals of a website/webpage.

Quickfire Keywords

Syntax - The structure or grammar for a high-programming language.
Opening tag - The beginning tag infront an html line of code.
Closing tag - The ending tag is at the end of an HTML line of code.
Content - What's written between the opening and closing tag.
Element - The entire line of code; the opening tag, closing tag and content.
External Styling - Linking an HTML document to a CSS file (separate from the HTML document).
Internal Styling - Writing CSS code in the same file as the HTML document.
Inline Styling - Using 'style' inside an html tag to change the properties of an element.
Let's begin with HTML, as it's the basic skeleton of most websites in the Internet.

HTML

htmL
In any sketchbook or program dedicated to multiple high level languages, the tag specifies the file you save the code as as html. Or, if you have multiple instances of programming languages in the same file, the tag helps classify which sections are in html (although that might be redundant, as it's better grouping code in one particular section rather than making multiple sections).

head
The head tag concerns the metadata of a website, the unstructured data in a website; things like keywords for the type of content inside the html site or the title of the tab. (Example: if your site is a business website, the head tag will include it's specificities, is it for a company or a standalone educational website?)

style
The style tag is a means to introduce CSS styling into your HTML document. Anything between the opening and closing tag are considered CSS code, this uses a different syntax than HTML (internal styling). This word can be put within a tag as well and can also be used for inline styling.

body
As the name suggests, this contains everything inside the HTML document. (Note: Styling an entire document would be easier if everything were to be inside this tag).

h1-h6
These are called Heading tags and in their most basic format highlight the headings of a webpage. The different font sizes are in descending order with the "h1" tag being the largest size. Every Heading tag is typically with a large font, this stands out from the body text and is styled 'bold'.

em
A format tag used to make the content within the tags (text-based content) in the italics style.

p
This is the paragraph tag. When used continously, the content between the tags are spaced with blank lines and treated like paragraphs.

marquee
A tag which gives an animation attribute to any content (text, image, video, etc...). The default settings is a slow horizontal panning of the content form right to left.

id
A unique label for an element, primarily for identification and also for inline styling. It can be given to one HTML element.

class
This is a kind of classification or grouping function, it's useful for external and internal styling and can change the attributes of multiple elements in an HTML document.

div
A tag which marks a section in an html document, it separates elements outside of the tag by containing it and allocate CSS styling to the element/elements inside using class or id attribute(s).

dl
A dl, is a description list and is used with dt and dd to create formatted text.

dt
This is called a 'Description Title', it introduces what the description dd will be about.

dd
This the 'Description Description'. Yes. Inside this tag, you can describe the subject written in the dt and it will leave a small indentation and then the following content added between the dd tags. Any images, videos or embedded links are considered.

h1 { font-family: "Anton", sans-serif; font-weight: 500; letter-spacing: 3; font-size: 35; }
This is an example of CSS code responsible for styling the h1 tag. Styling html tags are possible this way and are a direct alteration (what I consider 'indirect' is using class and ids). This overwrites or adds on to the properties of what the h1 tag provides by default. The next time a h1 tag is used again, it will have the following.

Breakdown


font-family: "Anton", sans-serif;
This changes the font of text in the HTML document when called. The font name is "Anton" and is wrapped in quotations (Note: Font names require the whitespace inbetween; like calling upon another family like "Comic Sans MS"), this is the first arguement taken by the property. The second arguement 'sans-serif' works as a keyword and specifies what kind of style the alphabets/characters the font would have (sans, cursive, monospace, etc.)

font-weight: 500;
A simple property that changes the thickness of the letters in the text. The higher the value, the bolder the letters.

letter-spacing: 3;
This property is self-explanatory and increases or decreases the spacing inbetween letters.

font-size: 35;
This changes the size of the font.


Example 2


.column { text-align: justify; float: left; width: 45%; padding: 20px; }
This is an example of a 'class'. It has properties that edits the space a content would be inside of. This can supply changes to multiple elements in an HTML document, keeping them in the same format/container. Especially important when keeping consistency in a website.

Breakdown


text-align: justify;
Changes the horizontal alignment of text with justify, center, left or right.

float: left;
This property can be similar to alignment types, with left or right but with an additional 'none' arguement. When content is added into an HTML doc, it takes up horizontal space along with it's vertical height due to it's default container. Float allows elements and text to wrap around the content. This kind of formatting property is useful in keeping relevant information next to each other and reducing extra white space.

width: 45%;
Changes the width of a content's container, anything exceeding the limit set by the user will be added to the next line.

padding: 20px;
Padding concerns the space between the container and the content inside it. It can be useful when spacing content next to other content.
